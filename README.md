# CMS_SpringBoot_Mybatis_Vue

풀스택으로 혼자 진행한 CMS(Content Mangement System) 프로젝트

회사에서 항상 Spring legacy 및 OracleDB만 사용하여
Spring Boot MySQL를 사용해보고자 시작한 프로젝트
